I'm trying to live by a set of values.

Ever since I read this, I've been thinking: what are the values I want to live by? Recently, I've been struggling. I'm not sure what I want to do or what direction I want to go in (quite the teenage dilemna, I'll admit). My emotions have been fluctuating and I'm trying to put a finger on something, something for stability. I just don't know... what.

After I read the manifesto (that I've linked), I see a way out: the very least I can do is live by a set of values. No matter what happens, I can live by a set of values that I believe in. And as I thought about it, I realized: I'm already doing this. I believe in these values, and I've been trying to live by them.

1. You're on your own, kid.

This is intentionally taken off from a song from Taylor Swift's Midnights, where she sings, "You're on your own, kid". When I heard it, I instantly knew that while it wasn't my favorite song off the album, it was the most meaningful to me, personally. Ultimately: I'm on my own. I'm responsible for my own success and failures. If I make a mistake, it's up to me to fix. If I want something, I gotta go get it myself. I can't wait around, waiting for something to come to me.

Currently, I've been thinking of this in terms of my indecision. If I want to make a change, I need to step up my game. No one's going to save me; it's up to me to finally decide to do something.

2. Things are easier than they seem.

This has become a pattern in my life. I have anxiety, but lately, I've been slowly getting over it by simply asking: am I blowing things out of proportion? I blow too many things out of proportion, when the solution is simple: just do it.

More importantly, if you make a mistake, you can fix it. There's always a alternate route. This has given me so much peace, just coming to terms with this fact.

That's it, so far. Thanks for reading.

How was this website created?

After I saw this video on creating a server from scratch in C, I decided to try it out for myself, but in Python. I ended up writing around 170 lines of (pretty bad) code, and I added a couple extra features:

  • Posts, baby! Yes, the posts are being stored in a folder on the backend and then being served on the frontend, using a custom templating engine (of sorts, haha).
  • Ability to read environment variables. Currently, this is being used to cURL the Spotify API (with an OAuth token) to get what I'm currently listening to.

So unecessary, but so fun. Everything was written from scratch; only default libraries in Python (json, http, mimetypes, os, socket, threading and urllib), were used.

The design is inspired by CSS Tricks. You'll noticed I used similar fonts, including:

  • BL Melody for the headings
  • Alegreya for the text

The source code is located on GitHub.
